## ZipCracker [Multi-Threaded]

Written in GoLang

### Install

```bash
git clone https://github.com/tahadostifam/ZipCracker
cd ZipCracker
go get # i used go-1.18 in writing this tool
```

### Usage

```bash
go run main.go --passwd=./files/passwd.txt --zip_file=./files/zip_file.zip --threads=2
```
